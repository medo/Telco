Predecting the 6th month¶

As the usage is linearly increasing we will predict the 6th month using linear regression.

In [31]:
cols = ["206_USAGE", "207_USAGE", "208_USAGE", "209_USAGE", "210_USAGE"]
train["211_USAGE"] = 0
train["tmp"] = ''
def predict(s, m):
    vals = map(int, s.split(",")[:-1])
    reg = LinearRegression()
    t = np.array(range(0, 5))
    t.shape = (len(t), 1)
    reg.fit(t, vals)
    return reg.predict(m)
for i in cols: train['tmp'] += train[i].astype('string') + ','
train['211_USAGE'] = train.apply(lambda row: predict(row['tmp'], 6), axis=1)
train = train.drop(["tmp"], axis=1)
